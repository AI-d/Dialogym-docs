## 핵심 요약 (Executive Summary)

Dialogym은 "대화도 훈련이 필요하다"는 철학을 기반으로 한 AI 기반 대화 훈련 플랫폼입니다.
헬스장에서 근육을 단련하듯 대화 능력을 체계적으로 훈련할 수 있는 안전한 공간을 제공합니다.
AI는 거울(Mirror), 코치(Coach), 훈련 파트너(Training Partner) 역할을 하며, 사용자가 실제 상황에서 자신 있게 소통할 수 있도록 돕습니다.

## 서비스 철학

### "대화도 훈련이 필요하다"

**핵심 통찰:**

현대인은 수많은 메시지와 연락 속에서 살아가지만, 연락은 단순히 정보 전달이 아닙니다.

* 직장에서는 **상사 보고** 한 문장에 따라 신뢰가 달라지고,
* 취업 준비생은 **면접 답변의 말투**에 따라 합격 여부가 갈리며,
* 부모와 자녀, 학부모와 선생님 간의 **작은 표현 차이**가 관계에 큰 영향을 미칩니다.
* 연인이나 친구와의 관계에서도 **"뭐라고 답장해야 하지?"**라는 고민은 끊임없이 발생합니다.

즉, **"무엇을 말할까"보다 "어떻게 말할까"가 더 중요한 시대**입니다.

그러나 학교나 직장 어디에서도 이런 **실제적인 대화 기술**을 체계적으로 배우기 어렵습니다.
결국 많은 사람들이 "적절한 말하기와 쓰기"를 혼자서 시행착오로 배우고 있습니다.

---

### Dialogym의 접근

**"대화는 근육과 같다"**

* 헬스장에서 근육을 단련하듯, 대화 능력도 훈련을 통해 강화됩니다.
* 반복적 연습으로 자연스러운 습관을 형성합니다.
* 안전한 환경에서 실수를 통해 배우고 성장합니다.

**Dialogym = Dialogue + Gym**

* Dialogym은 **대화를 위한 체육관**입니다.
* AI는 트레이너이자 훈련 파트너로서, 사용자의 대화 근육을 키워줍니다.
* 지속적인 훈련으로 실제 상황에서의 자신감을 향상시킵니다.

## 핵심 컨셉

### 1. Mirror (거울)

**AI는 사용자를 비추는 거울입니다.**

* 단순히 답을 대신하는 존재가 아님
* 사용자의 표현을 거울처럼 반영
* 개선점을 명확하게 보여줌

**예시:**
```
사용자: "음... 그... 보고드릴게 있어요"

AI 피드백 (거울):
- 추임새 3회 감지
- 불확실한 어미 ("~있어요" → "~있습니다")
- 시작이 망설임으로 보임

개선안:
"보고드릴 내용이 있습니다"
```

---

### 2. Coach (코치)

**AI는 단순 교정이 아닌, 성장을 돕는 코치입니다.**

* "왜 이 표현이 더 나은가" 설명
* 사용자의 습관과 톤을 학습
* 맞춤형 피드백 제공
* 점진적 개선 가이드

**예시:**
```
코칭 포인트:
1. 추임새가 많으면 확신이 없어 보입니다
2. "~있어요"보다 "~있습니다"가 더 공식적입니다
3. 시작을 명확하게 하면 신뢰감이 높아집니다

다음 훈련 목표:
- 추임새 2회 이하로 줄이기
- 공식 어미 사용 습관화
- 첫 문장 명확하게 시작하기
```

---

### 3. Training Partner (훈련 파트너)

**AI는 실제와 유사한 대화 상대입니다.**

* 상황별 역할 수행 (상사, 면접관, 부모 등)
* 실시간 음성 대화로 현장감 제공
* 안전한 환경에서 실수 허용
* 반복 훈련 가능

**예시:**
```
시나리오: 상사 보고

AI (상사 역할):
"프로젝트 진행 상황 보고해"

사용자:
"네, 현재 80% 완료했습니다"

AI:
"구체적으로 어떤 부분이 남았어?"

→ 실제 상황과 유사한 대화 흐름
→ 예상치 못한 질문 대응 연습
```

## 사용자 시나리오

### 1. 직장인 - 상사 보고

**상황:**
김대리는 내일 팀장님께 프로젝트 진행 상황을 보고해야 합니다.
말이 꼬이거나 추임새가 많아서 신뢰도가 떨어질까 걱정됩니다.

**Dialogym 사용:**
1. "상사 보고" 시나리오 선택
2. AI 팀장과 실전 대화 연습
3. 실시간 피드백 (추임새, 발화 속도)
4. 대화 종료 후 상세 분석 확인
5. 개선안 3가지 (간결/공손/명확) 비교
6. 다시 연습하며 개선

**결과:**
자신감 있게 보고 완료, 팀장의 긍정적 피드백 획득

---

### 2. 취준생 - 면접 연습

**상황:**
박학생은 다음 주 대기업 면접을 앞두고 있습니다.
압박 질문에 당황해서 제대로 답하지 못할까 봐 불안합니다.

**Dialogym 사용:**
1. "면접 연습" 시나리오 선택
2. AI 면접관과 실전 Q&A
3. 예상치 못한 질문에 대응 연습
4. 답변 명료성, 논리성 피드백
5. 반복 훈련으로 자신감 향상

**결과:**
면접 당일 침착하게 대응, 합격

---

### 3. 학부모 - 선생님 연락

**상황:**
이엄마는 아이의 학교생활에 대해 담임 선생님께 문의해야 합니다.
너무 격식 없으면 실례고, 너무 딱딱하면 차갑게 보일까 걱정됩니다.

**Dialogym 사용:**
1. "교사-학부모" 시나리오 선택
2. 적절한 톤 연습 (존중+친근함)
3. 문의 내용 전달 연습
4. 공손도, 명료성 피드백
5. 3가지 버전 비교 (간결/공손/따뜻)

**결과:**
선생님과 원활한 소통, 아이 문제 해결

---

### 4. 청년 - 연인 관계

**상황:**
정씨는 연인과 다툰 후 화해 메시지를 보내야 합니다.
너무 차갑지 않으면서도 진심을 담아 전달하고 싶습니다.

**Dialogym 사용:**
1. "연인 갈등" 시나리오 선택
2. 다양한 표현 방식 연습
3. 감정 톤 조절 피드백
4. 3가지 버전 (간결/공손/따뜻) 비교
5. 최적의 표현 선택

**결과:**
진심이 담긴 메시지로 화해 성공

---

### 5. MZ세대 - 어른과의 대화

**상황:**
최학생은 친척 어른들과 대화할 때 존댓말을 어떻게 써야 할지 모릅니다.
평소 친구들과는 반말만 쓰다 보니 격식 있는 표현이 어색합니다.

**Dialogym 사용:**
1. "세대 간 소통" 시나리오 선택
2. 존댓말, 예의 표현 연습
3. 상황별 적절한 표현 학습
4. 반복 훈련으로 자연스럽게 체화

**결과:**
명절이나 가족 모임에서 자신감 있게 대화

---

### 6. 사회초년생 - 전화 응대

**상황:**
신입사원 강씨는 업무 전화가 올 때마다 긴장됩니다.
갑자기 전화가 오면 무슨 말을 해야 할지 당황스럽습니다.

**Dialogym 사용:**
1. "업무 전화" 시나리오 선택
2. 전화 받기 구조 학습
   - 인사 → 소속/이름 → 용건 확인 → 마무리
3. 다양한 상황 대응 연습
4. 실시간 피드백으로 개선

**결과:**
전화 공포증 극복, 업무 능력 향상

## 차별성

### 1. 훈련 + 실행 루프

**기존 서비스:**
- 답변만 제공하고 끝
- 사용자는 복사해서 사용

**Dialogym:**
```
작성 → AI 피드백 → 수정 → 개선 확인 → 실행 준비
  ↑                                    ↓
  └────────── 히스토리 저장 ─────────────┘
```
- 지속적 개선 루프
- 성장 추적 가능
- 습관화 지원

---

### 2. 실전과 유사한 훈련

**기존 챗봇:**
- 텍스트 대화만 가능
- 정형화된 Q&A

**Dialogym:**
- **실시간 음성 대화** (GPT-4o Realtime API)
- 상황별 역할극 (상사, 면접관, 부모 등)
- 예상치 못한 질문 대응 연습
- 현장감 있는 훈련 환경

---

### 3. 맞춤형 피드백

**기존 서비스:**
- 일반적인 개선안만 제시
- 개인화 부족

**Dialogym:**
- **점수화된 객관적 피드백**
  - 발화 속도 (WPM)
  - 추임새 횟수
  - 공손도 (1~10)
  - 명료성 (1~10)
  - 종합 점수 (0~100)
- **3가지 개선안 제시**
  - 간결하게
  - 정중하게
  - 따뜻하게
- 사용자가 상황에 맞게 선택

---

### 4. 상황 맥락 기반

**범용 AI (ChatGPT):**
- 상황 특화 부족
- 매번 새로 설명 필요

**Dialogym:**
- **6가지 핵심 시나리오**
  1. 상사 보고
  2. 면접 연습
  3. 연인 갈등
  4. 부모님 연락
  5. 동료 협업
  6. 교사-학부모
- 상황별 최적화된 프롬프트
- 역할극 시뮬레이션

---

### 5. 성장 추적

**일회성 서비스:**
- 사용 후 기록 없음
- 성장 확인 불가

**Dialogym:**
- **히스토리 저장**
  - 과거 대화 기록
  - 점수 변화 추이
  - 개선 영역 분석
- **성장 리포트**
  - 주간/월간 통계
  - 강점/약점 분석
  - 다음 목표 제시

## 지향 가치

### 1. 실용성

**당장 오늘의 연락에 도움**

* 내일 상사 보고 → 오늘 연습
* 다음 주 면접 → 미리 준비
* 지금 답장 고민 → 즉시 해결

**즉시 적용 가능한 솔루션 제공**

---

### 2. 성장성

**반복 훈련을 통한 능력 향상**

* 처음: 추임새 10회, 점수 45점
* 1주 후: 추임새 5회, 점수 65점
* 1개월 후: 추임새 1회, 점수 85점

**점진적 발전 추적 및 동기 부여**

---

### 3. 개인화

**나와 내 상대방에 맞는 대화 코칭**

* 사용자 습관 학습 (자주 쓰는 표현, 말투)
* 상황별 최적화 (직장/면접/가족/연인)
* 개인 맞춤 피드백

---

### 4. 심리적 안전감

**실수해도 괜찮은 연습 공간**

* 판단 없는 환경
* 무한 반복 가능
* 부담 없는 훈련
* 자신감 향상
