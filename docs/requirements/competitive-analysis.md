* **팀원 전체**: 경쟁 서비스를 이해하고 차별점을 명확히 해야 하는 구성원

---

## 핵심 요약 (Executive Summary)

기존 대화 훈련 서비스들은 특정 영역(발표, 면접, 정서 교류 등)에 국한되어 있으며, 실생활 전반의 대화 상황을 포괄하지 못합니다.
Dialogym은 실시간 음성 대화, 점수화된 피드백, 상황별 시나리오, 성장 추적 기능을 통합하여 "실생활 대화 전반을 아우르는 AI 훈련 플랫폼"으로 차별화됩니다.

## 경쟁 서비스 분석

### 1. Poised

**주요 기능:**
- 실시간 발표/스피치 피드백
- 발표 중 말하는 속도, 추임새, 에너지 분석
- 프레젠테이션 최적화

**강점:**
- 실시간 분석 정확도 높음
- 발표/프레젠테이션 특화

**한계:**
- 발표·프레젠테이션에만 국한
- 일상 대화, 메시지 작성 커버 부족
- 대화 상대와의 역할극 불가

**Dialogym과의 차이:**
- Dialogym은 **발표 + 일상 대화 + 메시지**까지 확장
- 상황별 역할극 제공 (상사, 면접관, 부모 등)

---

### 2. Yoodli

**주요 기능:**
- 면접/발표 역할극
- AI가 면접관이 되어 질문
- 답변 분석 및 개선안 제공

**강점:**
- 면접 준비에 특화
- 구체적인 Q&A 연습 가능

**한계:**
- 면접·발표에 치중
- 실생활 대화 (가족, 연인, 동료) 약함
- 메시지 작성 지원 없음

**Dialogym과의 차이:**
- Dialogym은 **면접 + 일상 대화 + 메시지 초안 추천** 모두 지원
- 더 넓은 시나리오 커버

---

### 3. CAISY

**주요 기능:**
- 직무 시뮬레이션
- 비즈니스 상황별 대화 훈련
- 업무 커뮤니케이션 개선

**강점:**
- 직장 내 커뮤니케이션 특화
- 실무 중심 시나리오

**한계:**
- 비즈니스 상황에만 한정
- 개인 관계 (가족, 연인, 친구) 커버 안 함
- 대중성 낮음

**Dialogym과의 차이:**
- Dialogym은 **업무 + 개인 관계** 모두 대응
- 더 넓은 사용자층 타겟

---

### 4. Rocky.ai

**주요 기능:**
- 리더십/커뮤니케이션 훈련
- 경영진, 관리자 대상
- 팀 관리 대화 연습

**강점:**
- 리더십 개발에 특화
- 고급 커뮤니케이션 스킬

**한계:**
- 리더십에 초점, 대중성 낮음
- 일반 직장인, 학생 대상 아님
- 실생활 대화 부족

**Dialogym과의 차이:**
- Dialogym은 **리더십 + 일반 직장인 + 학생 + 개인** 모두 타겟
- 접근성 높음

---

### 5. Replika

**주요 기능:**
- 정서적 챗봇 동반자
- 일상 대화, 감정 교류
- 친구/연인 같은 AI

**강점:**
- 정서적 지원
- 자연스러운 대화

**한계:**
- 정서 교류 중심, 실용 대화 훈련 부족
- 피드백 기능 없음
- 성장 추적 없음

**Dialogym과의 차이:**
- Dialogym은 **정서 교류 + 실용 피드백 + 훈련** 결합
- 목적이 다름 (동반자 vs 트레이너)

---

### 6. Aimy (CoachHub)

**주요 기능:**
- 직장 내 갈등 역할극
- 어려운 대화 시뮬레이션
- 피드백 제공

**강점:**
- 직장 갈등 상황 특화
- 구체적 시나리오

**한계:**
- 직장에만 국한
- 가정, 연애, 학업 커버 안 함
- 범용성 낮음

**Dialogym과의 차이:**
- Dialogym은 **직장 + 가정 + 연애 + 학업** 모두 커버
- 훨씬 넓은 범위

---

### 7. ChatGPT 등 범용 AI

**주요 기능:**
- 범용 대화/문장 생성
- 질문하면 답변 제공
- 다양한 용도로 활용 가능

**강점:**
- 범용성 매우 높음
- 무한한 활용 가능성

**한계:**
- 특정 상황 최적화 부족
- 피드백 루프 없음
- 점수화 시스템 없음
- 상대방 스타일 학습 없음
- 성장 추적 불가

**Dialogym과의 차이:**
- Dialogym은 **대화 훈련 특화**
- 상황 맥락 기반 피드백
- 점수화 + 성장 추적
- 역할극 시뮬레이션

## 기존 플랫폼의 한계

### 1. 영역 특화의 함정

**문제:**
대부분의 서비스가 특정 영역에만 집중

```
Poised, Yoodli → 발표/면접
CAISY, Rocky.ai → 비즈니스/리더십
Replika → 정서 교류
```

**한계:**
- 일상 대화까지 확장 안 됨
- 사용자가 여러 앱 사용해야 함
- 데이터 분산, 성장 추적 어려움

**Dialogym 해결:**
- **하나의 플랫폼에서 모든 상황 커버**
- 업무, 면접, 가족, 연인, 친구 등 통합

---

### 2. 피드백 부재

**문제:**
Replika, ChatGPT 등은 답변만 제공

**한계:**
- 사용자가 왜 그 답변이 좋은지 모름
- 개선 방향 제시 없음
- 반복 학습 불가

**Dialogym 해결:**
- **점수화된 객관적 피드백**
- 구체적 개선 방안 제시
- 반복 훈련으로 성장

---

### 3. 지속성 부족

**문제:**
일회성 사용에 그침

**한계:**
- 히스토리 저장 없음
- 성장 추적 불가
- 습관화 어려움

**Dialogym 해결:**
- **히스토리 저장 + 성장 리포트**
- 점수 변화 추이 분석
- 습관화 지원 (리마인드, 챌린지)

---

### 4. 상황 맥락 부족

**문제:**
범용 AI는 매번 상황 설명 필요

**한계:**
- 시간 소모
- 최적화 부족
- 일관성 없음

**Dialogym 해결:**
- **상황별 최적화된 시나리오**
- 역할극 자동 설정
- 맥락 이해 기반 피드백

---

### 5. 실전 연습 부족

**문제:**
텍스트 기반 서비스는 실제 대화와 거리감

**한계:**
- 음성 대화 연습 불가
- 현장감 부족
- 실전 적응 어려움

**Dialogym 해결:**
- **실시간 음성 대화** (GPT-4o Realtime)
- 실제와 유사한 역할극
- 현장감 있는 훈련 환경

## Dialogym의 차별성

### 1. 실행 밀착형 루프

**기존 서비스:**
```
질문 → 답변 제공 → 끝
```

**Dialogym:**
```
작성 → AI 피드백 → 수정 → 개선 확인 → 실행 준비
  ↑                                       ↓
  └──────── 히스토리 저장 + 성장 추적 ──────┘
```

**차별점:**
- 단순 추천이 아닌 **훈련 + 실행 보조**
- 지속적 개선 루프
- 성장 가시화

---

### 2. 실시간 음성 대화

**기존 서비스:**
- 텍스트 기반 (Replika, ChatGPT)
- 또는 발표만 지원 (Poised)

**Dialogym:**
- **GPT-4o Realtime API 활용**
- 음성 → 음성 실시간 대화
- 상황별 역할극 (상사, 면접관, 부모 등)
- WebRTC 기반 안정적 통신

**차별점:**
- 실제와 가장 유사한 훈련 환경
- 예상치 못한 질문 대응 연습
- 현장감 극대화

---

### 3. 점수화된 피드백

**기존 서비스:**
- 정성적 피드백만 제공
- 또는 피드백 없음

**Dialogym:**
```
발화 속도 (30점)
  +
추임새 (20점)
  +
공손도 (25점)
  +
명료성 (25점)
  =
종합 점수 (100점)
```

**차별점:**
- **객관적, 정량적 평가**
- 개선 영역 명확히 파악
- 성장 추적 가능

---

### 4. 3가지 개선안

**기존 서비스:**
- 단일 개선안만 제시

**Dialogym:**
```
원본: "음... 그... 보고드릴게 있어요"

개선안:
1. 간결하게: "보고드릴 내용이 있습니다"
2. 정중하게: "보고드릴 사항이 있어 말씀드립니다"
3. 따뜻하게: "잠깐 시간 내주시면 보고드리고 싶은 내용이 있어요"
```

**차별점:**
- 사용자가 상황에 맞게 **선택 가능**
- 다양한 톤/스타일 학습
- 맥락 이해 능력 향상

---

### 5. 다양한 시나리오

**기존 서비스:**
- 1~2가지 상황만 지원

**Dialogym:**
- **6가지 핵심 시나리오**
  1. 상사 보고 (직장)
  2. 면접 연습 (취업)
  3. 연인 갈등 (개인 관계)
  4. 부모님 연락 (가족)
  5. 동료 협업 (직장)
  6. 교사-학부모 (교육)

**차별점:**
- **실생활 전 영역 커버**
- 하나의 플랫폼에서 모든 상황 대응
- 지속적 사용 유도

---

### 6. 성장 추적 시스템

**기존 서비스:**
- 히스토리 없음 또는 단순 저장

**Dialogym:**
```
히스토리:
- 과거 대화 전체 저장
- 점수 변화 추이 그래프
- 강점/약점 영역 분석

성장 리포트:
- 주간/월간 통계
- 가장 많이 개선된 영역
- 다음 훈련 목표 제시
```

**차별점:**
- **데이터 기반 성장 관리**
- 동기 부여
- 습관화 지원
