## 2. 주요 논의

### 2-1. 도메인 관리 방식

* **소유자 정보 변경(무료)**  
  * 동일 레지스트라(가비아) 내에서 **도메인 소유자 정보만** 변경.  
  * 왕택준(팀장) 계정 → 김경민 계정(소유자)로 업데이트, **등록 기간 유지**.  
  * 가비아 고객센터 통해 간단히 절차 안내 가능.

* **도메인 이전(Transfer)**  
  * 왕택준(팀장) 가비아 계정 → 김경민 가비아 계정으로 **완전 이전(관리권 전부 이관)**.  
  * 가비아 내부 이동이면 비교적 간단.

* **권장 진행**  
  * **가비아 고객센터(1544-4755) 직접 문의**가 가장 확실.  
  * 보통 필요한 서류:  
    * 현재 소유자(왕택준) 동의 + 신분증 사본  
    * 새 소유자(김경민) 신분증 사본  
    * 가비아 양도 신청서(양식 제공)

* **도메인 현황(결정)**
  * 경민님이 도메인을 신규로 결제 완료하여 소유/이전 이슈는 해소됨.
  * 후속: 네임서버/DNS 레코드 설정, 접근 권한 공유 및 인프라 문서에서 절차 추적.
