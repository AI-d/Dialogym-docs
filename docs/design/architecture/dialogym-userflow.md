## 핵심 요약 (Executive Summary)

Dialogym은 사용자가 AI와 역할극 방식으로 실전 대화 훈련을 한 후, 대화 종료 시 AI 코치가 전환되어 점수화된 피드백과 3가지 스타일의 개선안을 제공하는 대화 훈련 플랫폼입니다. 구독제 기반으로 대화 횟수, 대안 개수, 프로필 개수, 시나리오 개수를 제한합니다.

## 문서 개요

### 작성 배경
Dialogym 프로젝트의 사용자 여정을 명확히 정의하여, 개발 전 팀원 전체가 서비스의 핵심 흐름을 이해하고 일관된 사용자 경험을 제공할 수 있도록 작성되었습니다.

### 목적
- 사용자가 AI와 실전 대화 훈련을 하는 전체 여정 정의
- 대화 종료 후 AI 코치 전환 시점과 피드백 제공 방식 명시 
- 각 단계별 화면 구성과 핵심 기능 설명

### 핵심 철학
"대화도 훈련이 필요하다" - Dialogue + Gym = Dialogym

## 전체 플로우

```
1. 로그인/회원가입
   ↓
2. 시나리오 선택
   ├─ 2.1 기본 시나리오 선택
   └─ 2.2 사용자 생성 시나리오 선택
   ↓
3. 시나리오 생성 (2.2 선택 시)
   - 역할 입력
   - 상황 입력
   - AI 프롬프트 자동 생성
   ↓
4. AI와 실시간 역할극 대화 (훈련 모드)
   ├─ 4.1 기본 시나리오로 대화
   └─ 4.2 사용자 생성 시나리오로 대화
   ↓
5. 사용자가 대화 종료 클릭
   ↓
6. AI 코치 모드 전환 → 전체 대화 분석
   ↓
7. 점수화된 피드백 + 개선안 A,B,C 제공
   ↓
8. 사용자 선택/수정
   ↓
9. 최종안 저장
   ↓
10. 히스토리 조회 
```

### 2. 시나리오 선택

**6가지 핵심 시나리오**
1. 상사 보고 (직장)
2. 면접 연습 (취업)
3. 연인 갈등 (개인 관계)
4. 부모님 연락 (가족)
5. 동료 협업 (직장)
6. 교사-학부모 (교육)

**화면 구성**
```
┌─────────────────────────────────────┐
│  어떤 대화를 연습하고 싶으신가요?     │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │ 상사 보고                         │ │
│  │ 상사에게 요청사항을 전달하는 연습  │ │
│  │                      [선택하기]   │ │
│  └─────────────────────────────────┘ │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │ 면접 연습                         │ │
│  │ 면접관의 질문에 대답하는 훈련      │ │
│  │                      [선택하기]   │ │
│  └─────────────────────────────────┘ │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │ 연인 대화                         │ │
│  │ 갈등 상황에서 감정 표현 연습       │ │
│  │                      [선택하기]   │ │
│  └─────────────────────────────────┘ │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │ + 나만의 시나리오 생성하기         │ │
│  │   맞춤형 대화 상황을 만들어보세요  │ │
│  │                      [생성하기]   │ │
│  └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**기능**
- 시나리오 카탈로그 조회
- 시나리오 선택 시 상대방 프로필 자동 설정
- 시나리오별 설명 및 예시 제공
- 사용자 생성 시나리오 목록 조회 및 선택

### 3. 시나리오 생성

**기능**
사용자가 원하는 대화 상황을 직접 만들 수 있는 기능입니다.

**화면 구성**
```
┌─────────────────────────────────────┐
│ [←]  나만의 시나리오 만들기           │
├─────────────────────────────────────┤
│                                       │
│  시나리오 제목                        │
│  ┌─────────────────────────────────┐ │
│  │ 예: 거래처와 계약 협상            │ │
│  └─────────────────────────────────┘ │
│                                       │
│  AI의 역할                            │
│  ┌─────────────────────────────────┐ │
│  │ 예: 까다로운 구매 담당자          │ │
│  └─────────────────────────────────┘ │
│                                       │
│  상황 설명                            │
│  ┌─────────────────────────────────┐ │
│  │ 예: 가격 인하를 요구하는          │ │
│  │ 거래처 담당자와 협상하는 상황     │ │
│  │ 담당자는 직설적이고 압박적인      │ │
│  │ 태도를 보임                       │ │
│  └─────────────────────────────────┘ │
│                                       │
│  난이도 선택                          │
│  ○ 쉬움  ◉ 보통  ○ 어려움            │
│                                       │
│  카테고리                             │
│  ○ 직장  ○ 관계  ○ 가족  ○ 친구     │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │        [취소]      [생성하기]    │ │
│  └─────────────────────────────────┘ │
│                                       │
└─────────────────────────────────────┘
```

**입력 항목**
- 시나리오 제목: 사용자가 구분할 수 있는 간단한 제목
- AI의 역할: AI가 맡을 상대방의 역할 (예: 까다로운 고객, 권위적인 상사, 화난 연인 등)
- 상황 설명: 구체적인 대화 상황과 맥락 설명
- 난이도: 쉬움/보통/어려움
- 카테고리: 직장/관계/가족/친구

**시스템 처리**
1. 사용자 입력값 수집
2. AI 프롬프트 자동 생성
    - 역할과 상황을 바탕으로 AI의 행동 패턴 정의
    - 대화 스타일, 난이도에 맞는 반응 패턴 설정
3. 시나리오 DB 저장 (사용자 소유)
4. 즉시 대화 시작 가능

**제약사항**
- 구독 플랜에 따라 생성 가능한 시나리오 개수 제한
- Free: 1개, Premium: 10개, Pro: 무제한

### 4. AI와 실시간 역할극 대화 (훈련 모드)

**AI의 역할: Training Partner (훈련 파트너)**

#### 4.1 기본 시나리오로 대화

기본 제공되는 시나리오 선택 시:
- AI는 미리 정의된 역할과 프롬프트로 대화 시작
- 상사, 면접관, 연인 등 표준화된 역할 수행
- 일관된 품질의 대화 훈련 제공

#### 4.2 사용자 생성 시나리오로 대화

사용자가 직접 만든 시나리오 선택 시:
- AI는 사용자가 입력한 **역할**과 **상황**에 맞춰 대화 시작
- 사용자가 정의한 맥락과 난이도 반영
- 개인화된 대화 훈련 경험 제공

**공통 화면 구성**
```
┌─────────────────────────────────────┐
│ [←]  상사 보고                        │  ← 헤더
│ 권위적이고 직설적인 상사             │    (또는 사용자 생성 시나리오 제목)
│ 대화 시간: 00:03:45                  │
├─────────────────────────────────────┤
│                                       │  ← 대화 영역
│  ┌─────────────────────────────┐   │    (스크롤 가능)
│  │ AI(상사)                     │   │
│  │ 무슨 일이신가요?             │   │
│  └─────────────────────────────┘   │
│                                       │
│                                       │
│       ┌─────────────────────────┐   │
│       │ 사용자                   │   │
│       │ 팀장님, 내일 휴가        │   │
│       │ 가능하신가요?            │   │
│       └─────────────────────────┘   │
│                                       │
│  ┌─────────────────────────────┐   │
│  │ AI(상사)                     │   │
│  │ 갑자기? 이유는?              │   │
│  └─────────────────────────────┘   │
│                                       │
│                                       │
│       ┌─────────────────────────┐   │
│       │ 사용자                   │   │
│       │ 개인 사정이 있어서요     │   │
│       └─────────────────────────┘   │
│                                       │
│  ┌─────────────────────────────┐   │
│  │ AI(상사)                     │   │
│  │ 좀 더 구체적으로             │   │
│  │ 설명해 줄 수 있나요?         │   │
│  └─────────────────────────────┘   │
│                                       │
├─────────────────────────────────────┤
│ [실시간 피드백 - 접었다 펼 수 있음]  │  ← 피드백 영역
│ 추임새 2회 | 발화속도 120 WPM        │    (스와이프로 열고 닫기)
│ 침묵 5초                              │
├─────────────────────────────────────┤
│          ┌─────────────┐             │  ← 음성 입력 영역
│          │     🎤      │             │    (하단 고정)
│          │   탭하여    │             │
│          │   말하기    │             │
│          └─────────────┘             │
│                                       │
│              대화 종료             │  ← 대화 종료 버튼
└─────────────────────────────────────┘
```

**음성 전용 UI 요소**
- 마이크 버튼: 크고 명확한 원형 버튼 (직경 80px 이상)
- 탭하여 말하기: Push-to-Talk 방식 또는 자동 음성 감지
- 음성 인식 중 표시: 애니메이션 + 텍스트로 상태 표시
- 대화 내용은 실시간 STT로 텍스트 변환되어 화면에 표시

**모바일 최적화 요소** 
- 대화 영역은 전체 화면의 60-70% 차지
- 실시간 피드백은 하단에 접을 수 있는 패널로 구성
- 마이크 버튼은 하단 중앙에 크게 배치
- 터치 친화적인 버튼 크기 (최소 44x44px)
- 스와이프로 피드백 패널 열고 닫기 가능

**핵심 특징**
- **음성 전용 대화 방식**
- AI는 선택한 프로필의 역할(Role Play)만 수행
- 실시간 음성 대화 (GPT-4o Realtime API)
- WebRTC 기반 안정적 통신
- 음성 → STT → 화면에 텍스트로 표시
- 자연스러운 대화 흐름
- 예상치 못한 질문 대응 연습

**음성 입력 방식**
- Push-to-Talk: 마이크 버튼을 누르고 있는 동안 음성 인식

**대화 중에는 코치 피드백 없음**
- AI는 역할극에만 집중
- 실전과 같은 현장감 제공
- 사용자가 편안하게 연습할 수 있는 환경

**실시간 간단 피드백**
- 추임새 횟수 (음..., 그... 등)
- 발화 속도 (WPM)
- 침묵 시간
- 대화 중단 없이 화면에 가볍게 표시

**사용자 주도 종료**
- 사용자가 원할 때 [대화 종료] 버튼 클릭
- 연습이 충분하다고 느낄 때 자유롭게 종료

### 5. 대화 종료

**종료 방법**
1. [종료] 버튼 클릭
2. 종료 확인 알람탕

**종료 확인 알람탕**
```
┌─────────────────────────────────────┐
│                                       │
│       대화를 종료하시겠습니까?        │
│                                       │
│   AI 코치가 대화를 분석하고           │
│   개선안을 제공합니다                 │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │        [취소]        [종료]      │ │
│  └─────────────────────────────────┘ │
│                                       │
└─────────────────────────────────────┘
```

종료 버튼 클릭 시 AI가 역할극 모드에서 코치 모드로 전환됩니다.

### 6. AI 코치 모드 전환 → 전체 대화 분석

```
│  │ 간결하고 명료하게                │ │
│  │                                   │ │
│  │ "네, 현재 80% 완료했습니다.      │ │
│  │  테스트와 문서 작업이            │ │
│  │  남아있습니다."                  │ │
│  │                                   │ │
│  │ ✓ 추임새 제거                    │ │
│  │ ✓ 구체적 표현                    │ │
│  │ ✓ 명확한 문장 구조               │ │
│  │                                   │ │
│  │  [이 안 사용하기]                │ │
│  └─────────────────────────────────┘ │
│                                       
├─────────────────────────────────────┤
└─────────────────────────────────────┘
```

**핵심 특징**
- AI가 전체 대화 히스토리를 리뷰
- 점수화 룰 적용: 발화 속도, 추임새, 공손도, 명료성
- 대안 3가지 제시: 간결/공손/따뜻 스타일
- 각 대안별 개선점 명시

### 9. 최종안 저장

**저장 데이터**
```
- 세션 ID
- 선택한 시나리오 (기본 또는 사용자 생성)
- AI 프로필 (상사, 면접관, 연인 등)
- 대화 시작/종료 시간
- 전체 대화 내용 (transcript)
- 실시간 피드백 데이터 (추임새, 발화 속도, 침묵)
- 최종 점수 및 세부 점수
- 개선 포인트
- 제시된 대안 A, B, C
- 사용자가 선택한 최종안
```
